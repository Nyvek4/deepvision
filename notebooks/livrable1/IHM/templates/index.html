Comment ça fonctionne

Notre modèle de deep learning analyse les caractéristiques de votre image pour déterminer son authenticité.

1. Uploader une image

Sélectionnez ou glissez-déposez une image depuis votre appareil. Nous supportons les formats JPG, PNG et WEBP.

2. Analyse par IA

Notre modèle CNN analyse les caractéristiques de l'image pour détecter si c'est une photo réelle ou non.

3. Obtenez les résultats

Recevez une analyse détaillée avec les probabilités associées à chaque classe.

Notre modèle IA

Nous utilisons une architecture ResNet50 pré-entraînée sur ImageNet, fine-tunée pour la classification photo/non-photo.

Le modèle a été entraîné sur un dataset de +40,000 images avec une précision de validation de 98,66% sur plus de 6000 images de tests (15%).

TensorFlow Keras ResNet50 Transfer Learning
